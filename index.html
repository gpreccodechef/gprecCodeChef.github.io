CP Roadmap

Programming Languages:

C++

  • Fastest executing programming language for CP
  • Incredible STL support
  • Less boilerplate code

Java

  • Fast execution time but slower than C++
  • More boilerplate code

Python

  • Good for beginners as they can stress more on logic than on engineering
  • VERY SLOW EXECUTION TIME!
  • NOT PREFERABLE FOR ADVANCED CP!

    • Logic Building & Math:

    You should have the knowledge of atleast one programming language to step into CP. That programming language will be a tool for you. It can be used for converting the thought process into the code.
    Sometimes you may get the right logic, but you cannot implement it. Understand the Concepts like Time Complexity and Space Complexity. Solve the problems with the implementation tags.
    Some sample problems: Petya and Strings, Young Physicist, Helpful Maths.

    DS & Algo:

    Basic DS & Algo:

    Learning data structures and algorithms allow us to write efficient and optimized computer programs. Implement these basic DS & Algo on some of the questions on the various coding platforms like CodeChef/Codeforces.
    Some important topics to learn are Sorting, Greedy, Binary Search, Counting Sort and Fast Exponentiation.
    Some Sample Problems: Guessing Game, Chefina and Swaps Doctor Chef.

    Advanced DS & Algo:

    If you feel that you are confident and perfect on basic DS & Algo, then you may look at the Advanced Concepts on DS & Algo like Segment Trees, Fenwick Trees, Graph Algorithms, Dynamic Programming.
